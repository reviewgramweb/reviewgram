Помощь по вводу голосом

Общие рекомендации

Старайтесь произносить слова чётко и отчётливо, без ошибок. Используйте паузы не менее трети секунды между словами, чтобы улучшить качество распознавания. Так как Reviewgram использует распознавание слов на английском языке, потренируйте свой акцент, чтобы улучшить качество распознавания. Можно использовать Google Translate как пример распознавания.

Таблицы замены

В настройках репозиториев существует возможность задать таблицы замены, которые позволяют исправлять локальные ошибки рапознавания. Если вы замечаете, что распознавание ошибок сликом часто ошибается, используйте данные таблицы, чтобы заменить одно слово на другое в процессе распознавания. Данные таблицы применяются до этапа замены слов в рамках распознавания языка программирования, при этом сравнение является нечётким (слова должны быть похожи на 70%, чтобы быть заменены), поэтому стоит быть осторожным при использовании данных таблиц.

Таблицы замены для языка Python

Для языка Python определённые свои таблицы, которые позволяют упростить ввод определённых модулей, символов и данных. При этом реализована автозамена похожих идентификаторов, из исходного кода при вводе.
